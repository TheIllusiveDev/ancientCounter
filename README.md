# 🛸 Stargate Ancient Countdown Timer

An open-source countdown timer styled after the **Lantean (Ancient) consoles** seen in *Stargate SG-1* and *Stargate Atlantis*. Built with C# and .NET 9, this project aims to bring a piece of the Pegasus Galaxy to your desktop.

> [!CAUTION]
> **Font Required:** This application requires the **Ancient (Lantean) font** to be installed on your system. Without it, the interface will default to standard system text and lose its authentic look.

---

## ✨ Features
* **Lantean Aesthetic:** Specifically designed to mimic Ancient technology interfaces.
* **Custom Intervals:** Set your own countdown timer duration via the GUI.
* **Modern Framework:** Built on **.NET 9** for high performance.
* **Open for Collaboration:** This is a fan-made project; feel free to fork, modify, and improve the code!

## 🛠️ Prerequisites
To run or develop this project, you will need:

1.  **.NET 9 SDK:** [Download here](https://dotnet.microsoft.com/download/dotnet/9.0)
2.  **IDE:** [Visual Studio 2022](https://visualstudio.microsoft.com/vs/) or [JetBrains Rider](https://www.jetbrains.com/rider/).
3.  **Ancient Font:** Ensure you have the Lantean `.ttf` or `.otf` installed in your system fonts folder.

## 🚀 Getting Started
1. **Clone the Repository:**
    ```bash
    git clone [https://github.com/YourUsername/YourProjectName.git](https://github.com/YourUsername/YourProjectName.git)
    ```
2. **Install the Font:** Open your downloaded Ancient font file and click **Install**.
3. **Open the Solution:** Launch your IDE and open the `.sln` file.
4. **Build & Run:** Hit `F5` to compile and start the timer.
